# scoop-bucket

Scoop bucket for tools by George Mandis.

## Install

```powershell
scoop bucket add georgemandis https://github.com/georgemandis/scoop-bucket
scoop install georgemandis/whereami
```

## Available Manifests

| Manifest | Description |
|----------|-------------|
| [whereami](https://github.com/georgemandis/whereami) | Get your current location from the command line |
